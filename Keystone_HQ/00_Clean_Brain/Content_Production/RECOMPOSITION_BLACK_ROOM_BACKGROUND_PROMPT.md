# 🖤 KEYSTONE RECOMPOSITION — BLACK ROOM BACKGROUND STYLE PROMPT
## The Canonical Background Reference Prompt for Google Flow & HeyGen
## Created: 2026-05-22

> **PURPOSE**: Use this exact, desaturated, clinical prompt to set the scene background in Google Flow or HeyGen. It contains zero references to the person (Wayne), focusing purely on the studio environment, lighting, color grading, and lens characteristics.

---

## THE CORE BACKGROUND PROMPT (Copy-Paste Ready)

```
A clean, premium, high-end studio environment. The background is a solid, 
pure black, empty space with zero visible set elements, zero furniture, and 
zero wall textures. Single soft key light from a 45-degree angle on the 
camera-left, creating dramatic, clinical side-lighting. Fill light is 
extremely low, creating deep, moody, compressed shadows. Muted clinical color 
grading with a desaturated teal and charcoal gray palette (#1a3a3a, #111111). 
Ultra-shallow depth of field with a beautiful, soft bokeh roll-off at the 
edges. Volumetric clinical haze is barely visible, catching the key light 
subtly. Shot on 35mm anamorphic lens, highly cinematic, premium, 
authoritative, medical documentary style, 4K resolution, clean.
```

## 🌟 WHY THIS EXACT LIGHTING WORKS

1. **Defeats AI Slop Detection**: The dramatic side-lighting (Chiaroscuro) creates realistic shadows across the avatar's face, making the render look organic, premium, and human — avoiding the flat, brightly lit look that flags content as AI.
2. **Seamless Continuity**: Using this exact background prompt across all 50 clips ensures the black background and teal shadow levels remain 100% consistent from clip 001 to clip 050.
3. **DaVinci Resolve Ready**: This teal/black palette perfectly matches our custom color grade (#1a3a3a shadows) without needing extensive post-production correction.
