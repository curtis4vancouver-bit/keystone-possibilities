<!-- CONTEXT: Platform Standards / 1 · YouTube -->
## 1 · YouTube

<!-- CONTEXT: Platform Standards / Long-Form Video -->
### Long-Form Video

| Element               | Specification                                        |
|-----------------------|------------------------------------------------------|
| Resolution            | 1920×1080 (1080p) minimum, 3840×2160 (4K) preferred |
| Aspect ratio          | 16:9                                                 |
| Frame rate            | 24fps or 30fps (consistent within series)            |
| Duration              | 8-15 minutes (sweet spot for algorithm)              |
| Thumbnail             | 1280×720, <2MB, JPG or PNG                           |
| Title length          | 50-60 characters (front-load keywords)               |
| Description           | 200+ words, timestamps, links, disclaimer            |
| Tags                  | 15-30 relevant tags, mix broad and specific          |
| Cards                 | Add at natural transition points                     |
| End screen            | Last 20 seconds — subscribe + next video             |
| Chapters              | Required — minimum 3 chapters with timestamps        |

<!-- CONTEXT: Platform Standards / YouTube Shorts -->
### YouTube Shorts

| Element               | Specification                                        |
|-----------------------|------------------------------------------------------|
| Resolution            | 1080×1920 (9:16)                                     |
| Duration              | 30-58 seconds (55 seconds optimal)                   |
| Title length          | 40 characters max (less is more)                     |
| Hashtags              | 3-5 in description (`#shorts` NOT required anymore)  |
| Hook                  | First 2 seconds must stop the scroll                 |
| CTA                   | "Full video on the channel" — drive to long-form     |
| Text overlays         | Large, high contrast, safe zone centred               |

<!-- CONTEXT: Platform Standards / YouTube Ambient ([[music|Music]] Brand) -->
### YouTube Ambient (Music Brand)

| Element               | Specification                                        |
|-----------------------|------------------------------------------------------|
| Resolution            | 1920×1080 or 3840×2160                               |
| Duration              | 1-4 hours                                            |
| Visual                | Slow-moving abstract, waveform, or static album art  |
| Description           | Track name, frequency, Spotify link, minimal text    |
| Title format          | "[Track Name] · [Frequency] · [Duration]"            |
| No end screens        | Ambient videos should loop/fade naturally             |

<!-- CONTEXT: Platform Standards / YouTube SEO Rules -->
### YouTube SEO Rules

1. **Title:** Primary keyword within first 5 words
2. **Description:** First 150 characters are critical (shown in search)
3. **Thumbnail:** Must be custom — never auto-generated
4. **Tags:** Include brand name, topic, and long-tail variations
5. **Playlist:** Every video belongs to at least one playlist
6. **Pinned comment:** Summary or additional context

### Video Specifications

| Element               | Specification                                        |
|-----------------------|------------------------------------------------------|

<!-- CONTEXT: Platform Standards / TikTok Content Strategy -->
### TikTok Content Strategy

- **Protocol brand only** (Possibilities audience isn't on TikTok)
- **Repurpose YouTube Shorts** with TikTok-native editing
- **Hook formula:** Controversial statement → Explanation → CTA
- **Never:** Mention YouTube in the video (TikTok deprioritizes cross-platform CTAs)
- **Comment replies:** Film response videos to top comments

## 4 · Website / Blog

<!-- CONTEXT: Platform Standards / Blog Post Standards -->
### Blog Post Standards

| Element               | Specification                                        |
|-----------------------|------------------------------------------------------|
| Word count            | 1,500-3,000 words (SEO sweet spot)                  |
| Title (H1)            | One per page, includes primary keyword               |
| Meta description      | 150-160 characters, includes primary keyword         |
| Headers               | H2 for sections, H3 for subsections, logical hierarchy|
| Images                | 3-5 per post, alt text required, compressed           |
| Internal links        | 3-5 to relevant existing content                     |
| External links        | 1-3 to authoritative sources (studies, government)   |
| CTA                   | Clear CTA in final section                           |
| Schema markup         | Article, FAQ, or HowTo where applicable              |
| URL slug              | Short, keyword-rich, no dates                        |

<!-- CONTEXT: Platform Standards / Blog SEO Checklist -->
### Blog SEO Checklist

- [ ] Primary keyword in title, first paragraph, and at least one H2
- [ ] Meta description written (not auto-generated)
- [ ] All images have descriptive alt text
- [ ] Internal links to 3+ related posts
- [ ] At least one external authority link
- [ ] Mobile-responsive formatting verified
- [ ] Table of contents for posts >2,000 words
- [ ] FAQ section with schema markup where relevant
- [ ] Published to sitemap and submitted to Google Indexing API

<!-- CONTEXT: Platform Standards / Website Visual Standards -->
### Website Visual Standards

- **Possibilities site:** Blue/gold palette, clean professional layout
- **Recomposition site:** Dark theme, gold accents, premium aesthetic
- **Both sites:** Fast load times (<3s), mobile-first design, no layout shift

<!-- CONTEXT: Platform Standards / 8 · Cross-Platform Content Repurposing Flow -->
## 8 · Cross-Platform Content Repurposing Flow

```
Blog Post (1,500+ words)
    ├──→ YouTube Long-Form (8-15 min script from blog)
    │       ├──→ YouTube Short (key moment, 30-55 sec)
    │       ├──→ Instagram Reel (recut for IG, 15-60 sec)
    │       ├──→ TikTok (Protocol only, native edit)
    │       └──→ Facebook Native Video (full or trimmed)
    ├──→ Instagram Carousel (5-7 slides from blog key points)
    ├──→ Facebook Post (blog summary + link)
    └──→ Google Post (Possibilities — snippet + link)
```

<!-- CONTEXT: Platform Standards / Repurposing Rules -->
### Repurposing Rules

1. **Never post identical content** across platforms — adapt for each audience
2. **Platform-native** always — upload directly, don't share links between platforms
3. **Stagger releases** — blog first (SEO indexing), then video, then social
4. **Track which content came from which source** in `RECENT_CONTENT.json`
