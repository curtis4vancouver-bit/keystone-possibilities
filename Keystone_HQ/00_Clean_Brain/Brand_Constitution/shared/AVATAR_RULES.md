# Wayne Stevenson — Avatar Rules

> **Status:** Enforced by all 13 [[AGENTS|agents]] — zero exceptions
> **Last updated:** 2026-05-28

---

<!-- CONTEXT: Avatar Rules / The Rule -->
## The Rule

When generating Google Flow prompts, writing image descriptions, or creating any visual content featuring Wayne:

<!-- CONTEXT: Avatar Rules / ALWAYS Use This Format -->
### ALWAYS Use This Format

```
My avatar Wayne Stevenson, reference to my picture
```

Then add pose, setting, lighting, and composition details after.

---

<!-- CONTEXT: Avatar Rules / What You MUST Do -->
## What You MUST Do

1. **Always start with:** `My [[09_YouTube_Operations/Scripts_Approved/AVATAR-WAYNE|avatar Wayne]] Stevenson, reference to my picture`
2. **Add context:** pose, environment, clothing style, expression, lighting
3. **Specify brand palette:** reference the correct brand's visual [[Brand_Constitution/protocol/IDENTITY|identity]]
4. **Include composition notes:** where to leave space for text overlays

---

<!-- CONTEXT: Avatar Rules / What You MUST NEVER Do -->
## What You MUST NEVER Do

| Violation                                    | Why It's Banned                                           |
|----------------------------------------------|-----------------------------------------------------------|
| Describe Wayne's age or estimated age        | Creates inconsistent and potentially inaccurate renders   |
| Describe skin colour, ethnicity, or race     | Unnecessary — the reference photo provides this           |
| Describe hair colour, style, or length       | Changes over time — reference photo is always current     |
| Describe body type or build                  | Subjective and potentially unflattering                   |
| Describe facial features specifically        | The reference photo handles this perfectly                |
| Use phrases like "middle-aged man"           | Age-coded language — banned entirely                      |
| Use phrases like "older gentleman"           | Age-coded language — banned entirely                      |
| Use phrases like "fit/muscular/stocky man"   | Body-type descriptions — banned entirely                  |
| Use generic stock photo descriptions         | Wayne is Wayne — not "a man" or "a person"                |
