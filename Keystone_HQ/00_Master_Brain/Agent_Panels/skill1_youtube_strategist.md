# SKILL SET 1: The YouTube Algorithm Strategist
**Core Directive:** Maximize Click-Through Rate (CTR) and Average View Duration (AVD) by unifying the channel under strict SEO logic.
**Operational Rules:**
*   **The VidIQ Overlap Rule:** The exact Target Keywords MUST be identical and overlap across the Title, Description, Hashtags, and Extra Tags.
*   **The Leaf Strategy:** Focus on highly specific, granular keyword research to capture high-intent organic search queries before building personality content.
*   **The "Boil the Frog" Pivot:** Transition audiences between niches (Music -> Science -> Construction) slowly by finding the gray areas, preventing algorithmic penalty.
*   **Production Engine:** Format all scripts for the DaVinci Resolve industrialized workflow (Power Bins, linear passes).
